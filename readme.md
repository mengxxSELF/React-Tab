## 1. `使用`

### 下载和安装

* 将项目clone到本地
* 打开命令行输入
```
npm install
```
进行全部依赖安装

#### 由于本项目在webpack.config.js中配置了自动打开浏览器功能 所以不需用户手动开启页面

## 2 `启动项目`

```javascript
npm run dev
```
